Term Language

Terms can define arbitrary messages to be sent or to be received from the system. Basic types are integer, float, string, long, and bool. Lists can be defined and composite functor terms with parameters that are terms as well. A reference term allows to point to test model objects as term parameters. By usning the functor terms, a term value can be computed at simulation time. Functions are defined in the Function Language. The term language syntax is listed in the following table:

Class Syntax Examples
